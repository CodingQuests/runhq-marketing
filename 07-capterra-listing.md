## 1. Category research

Capterra requires you to pick categories. Best-fit options to request (pick 1 primary + 2–3 secondary):

| Category | Fit | Why |
|----------|-----|-----|
| **Software Development** | Primary | Closest to core: tools for building/shipping software. |

## 2. Listing copy

### Product name
RunHQ

### Tagline (short)
The operations layer for AI coding agents.

### Short description (≈50 words)
RunHQ turns product feedback into agent-ready coding tasks, reviewed pull requests, preview environments, and shipped updates. Built for technical founders and AI-native teams using tools like Claude Code, Cursor, and Codex, it runs the workflow around your coding agents — so you ship faster and safely.

### Long description (≈150 words)
AI coding agents can write code faster than teams can review, test, and ship it. RunHQ fixes that bottleneck.

RunHQ is the operations layer for AI coding agents. It takes you from raw product feedback to shipped update: paste a bug report or user request, and RunHQ turns it into an agent-ready task. Your agent writes the code, RunHQ reviews the resulting pull request, spins up a preview environment to test it, and gives you a safe path to ship.

Instead of duct-taping project management, GitHub, and deployment tools around your agents, RunHQ runs the whole loop in one place — and works with the stack you already use: Claude Code, Cursor, Codex, GitHub, Vercel, and Linear.

Built for technical founders, indie hackers, and small AI-native teams who want the speed of AI coding without losing control of quality or shipping safety.

### Feature bullets (verify each ⚠️)
- Turn product feedback into agent-ready coding tasks
- Automated review of agent-generated pull requests
- Preview environments for every change
- Safe, repeatable path from agent output to production
- Works with Claude Code, Cursor, Codex, Devin
- Integrates with GitHub, Vercel, and Linear

### Key benefits (outcome-framed)
- Ship AI-generated code faster, without skipping review
- Stop manually translating feedback into dev tasks
- One workflow instead of five disconnected tools
- Built for small teams — no enterprise overhead
