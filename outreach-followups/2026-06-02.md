## Objection quick-responses

| Objection | Response |
|-----------|----------|
| "I already use Linear + GitHub." | RunHQ sits *across* them — it automates the agent loop they don't cover: feedback triage → agent task → reviewed PR → preview. |
| "Isn't this just ClickUp/PM software?" | PM tools track human tickets. RunHQ operates the *agent* loop and produces real code artifacts — tasks, PRs, previews. |
| "Why not just CrewAI / build my own?" | Those build agents. RunHQ runs the *workflow around* them — you keep your agents, we run the feedback→shipped process. |
| "We're too small." | Small AI-native teams feel the review/ship bottleneck hardest — that's exactly the ICP. |
| "Does it ship on its own?" | No — a human reviews and approves every PR. It removes the busywork around the loop; you stay in control of what ships. |
