# RunHQ — 15-Second Homepage Video

> Goal: show the **core workflow in one breath** — feedback → reviewed PR → shipped. No narration needed (most autoplay muted); on-screen text carries it. Also doubles as the Meta ad creative.
> ⚠️ Shot list assumes the real product UI. Adjust to match actual screens.

---

## The one idea
**Messy feedback goes in → a reviewed, shippable PR comes out — automatically.**

## Script (15 seconds, ~6 shots)

| Time | Visual | On-screen text | Notes |
|------|--------|----------------|-------|
| 0:00–0:02 | A real-looking user message / bug report (Slack or support ticket) | "A user reports a bug." | Hook with the pain instantly. |
| 0:02–0:04 | Paste it into RunHQ → it becomes a clean, structured task | "RunHQ turns it into an agent-ready task." | Show the transform animation. |
| 0:04–0:07 | Coding agent (Claude Code / Cursor) picks it up, code appears | "Your agent writes the fix." | Fast-forward feel. |
| 0:07–0:10 | RunHQ review view: PR with structured review/checks passing | "RunHQ reviews the PR." | The differentiator — linger here. |
| 0:10–0:12 | Preview environment URL opens, bug is fixed | "Preview it live." | Proof it works. |
| 0:12–0:14 | Merge / ship animation, green checkmark | "Ship it. Safely." | Payoff. |
| 0:14–0:15 | RunHQ logo + tagline | "RunHQ — the operations layer for AI coding agents." | End card / CTA. |

## End-card CTA options
- "Start free → runhq.io"
- "From feedback to shipped. runhq.io"

## Production notes
- **Format:** export 16:9 (homepage), 1:1 and 9:16 (Meta/Reddit/Stories). Same cut, reframed.
- **Captions:** burn in the on-screen text — assume muted autoplay.
- **Pace:** quick cuts; the *speed* is the message (the whole loop in 15s mirrors the product promise).
- **Music:** subtle, modern, low — or none for the muted-autoplay version.
- **Tooling to record:** screen capture the real product (Screen Studio / Loom / OBS), then edit in CapCut / Premiere / Descript. Keep raw clips for re-cuts.
- **Realism:** use a believable bug + real-looking UI. Polished-but-real beats slick-but-fake for this technical audience.

## Alt 15-sec angle (if "feedback→PR" UI isn't ready to film)
Open on: *"Your AI agent finished the code."* → cut to a pile of unreviewed PRs → *"Now who reviews, tests, and ships it?"* → RunHQ handles review + preview + ship → end card. Same length, leans on Angle B.
